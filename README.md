# AI Virtual Staging & Smart Interior Toolkit

AI Virtual Staging & Smart Interior Toolkit is an open-source prompt generator and workflow toolkit for AI-powered real estate virtual staging, renovation visualization, and smart interior concept design.

This project helps real estate agents, interior designers, renovation companies, smart home builders, and non-technical users create structured AI prompts without advanced prompt engineering knowledge.

---

## Live Usage

You can use this project in two ways:

1. Open `index.html` directly in a browser.
2. Host the repository with GitHub Pages and use it as a simple web app.

No backend server is required.

---

## What This Tool Does

The included web-based prompt generator helps users create structured prompts for:

- Empty room conversion
- Real estate virtual staging
- Renovation before/after visualization
- Smart lighting concept design
- Luxury interior concept generation
- Home cinema room concept generation
- Therapy room concept generation

Users can select a room type, output goal, style, lighting mood, and constraints. The tool then generates a reusable prompt that can be copied and used in AI image generation tools.

## How to Use

### Option 1. Use Locally

1. Download this repository.
2. Open `index.html` in your browser.
3. Select options.
4. Click `Generate Prompt`.
5. Copy the generated prompt.

### Option 2. Use with GitHub Pages

1. Go to repository settings.
2. Open `Pages`.
3. Set source to `Deploy from a branch`.
4. Select `main` branch and root folder.
5. Save.
6. GitHub will provide a public web URL.
