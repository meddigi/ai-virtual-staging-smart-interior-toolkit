# Contributing Guide

Thank you for your interest in contributing to this project.

AI Virtual Staging & Smart Interior Toolkit is an open-source project for real estate virtual staging, renovation visualization, and smart interior concept design.

## How You Can Contribute

You can contribute by:

- Adding new virtual staging prompts
- Improving smart interior design prompts
- Translating prompts into other languages
- Adding workflow examples
- Sharing real estate listing copy examples
- Creating smart lighting and smart home scene prompts
- Reporting unclear, unrealistic, or low-quality prompt results
- Suggesting improvements for real estate and interior design use cases

## Contribution Guidelines

Please keep all contributions:

- Practical
- Realistic
- Easy to understand
- Useful for real estate, interior design, renovation, and smart interior workflows
- Free from private client information
- Free from copyrighted images or confidential project data

## Recommended Prompt Format

When adding a new prompt, please use this structure:

```markdown
# Prompt Title

## Purpose

Briefly explain what this prompt is for.

## Prompt

Write the full prompt here.

## Notes

Add usage tips, limitations, or recommended settings.
```

## How to Contribute

1. Fork this repository.
2. Create a new branch.
3. Add or improve prompt templates, workflows, examples, or documentation.
4. Submit a pull request.
5. Explain what you changed and why it is useful.
