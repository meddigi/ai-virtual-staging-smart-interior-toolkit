# Real Estate Virtual Staging Workflow

## Step 1. Analyze the Original Photo

Check the following:

- Room type
- Lighting condition
- Clutter level
- Existing furniture
- Wall and floor condition
- Window location
- Real estate listing purpose

---

## Step 2. Choose the Output Goal

Select one:

- Empty room conversion
- Modern staging
- Luxury staging
- Smart interior concept
- Renovation before/after concept

---

## Step 3. Apply the Prompt

Use a prompt from the `prompts` folder.

Recommended examples:

- `empty-room-conversion.md`
- `smart-interior-lighting.md`
- `luxury-living-room.md`

---

## Step 4. Review the Result

Check:

- Is the original room structure preserved?
- Are windows and doors unchanged?
- Is the image realistic?
- Is the space suitable for a property listing?
- Are there any AI artifacts?
- Does the result mislead the buyer?

---

## Step 5. Create Listing Copy

Use the final image to create:

- Real estate description
- SNS caption
- Renovation proposal text
- Before/after explanation
